# スミレ請求書 🌸

**登録不要・完全無料・データはどこにも送信されない**インボイス制度対応の請求書ジェネレーターです。

👉 **今すぐ使う:** https://taku0826.github.io/sumire-invoice/

## 特徴

- ✍️ **リアルタイムプレビュー** — 入力した内容がその場で請求書に反映されます
- 🖨️ **印刷 / PDF保存** — ブラウザの印刷機能でそのままきれいなPDFに
- 🧾 **インボイス制度対応** — 適格請求書発行事業者の登録番号(T+13桁)に対応
- 💴 **税率別の消費税計算** — 10% / 8%(軽減税率)を品目ごとに設定、軽減税率注記も自動表示
- 🔴 **朱印スタンプ** — 発行者の姓から朱印風のハンコを自動生成
- 🔒 **完全ローカル動作** — サーバーに一切データを送信しません。入力内容はあなたのブラウザの中だけ

## 使い方

1. ページを開く
2. 発行者・宛先・品目を入力する
3. 「印刷 / PDF保存」ボタンを押す

それだけです。アカウント登録も、メールアドレスの入力も要りません。

## 仕組み

単一のHTMLファイルだけで動作します。ビルドなし、依存ライブラリなし。
`index.html` をダウンロードしてオフラインで使うこともできます。
